API Endpoints

  • GET /api/users - Get all users
  • GET /api/users/<id> - Get single user
  • POST /api/users - Create user
  • PUT /api/users/<id> - Full update user
  • PATCH /api/users/<id> - Partial update user
  • DELETE /api/users/<id> - Delete user

When do we use DELETE method?

Use DELETE when you want to remove an existing resource permanently from the system, such as deleting a user, order, or product that is no longer needed.
